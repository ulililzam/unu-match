üìê Mathematical Formulas

1. Feature Normalization

$$x_{normalized} = \frac{x - x_{min}}{x_{max} - x_{min}}$$

Scales all features to [0, 1] range for consistent processing

2. Random Forest Prediction

$$P(y=c|X) = \frac{1}{T}\sum_{t=1}^{T} I(h_t(X) = c)$$

Where T = 300 trees, ht = prediction from tree t, I = indicator function

3. Weighted Voting System

$$P_{weighted}(c) = P(c) \times (1 + w \times \alpha)$$ $$w = \sum_{i=1}^{n} f_i \times imp_i$$

w = feature alignment weight, Œ± = 0.15 boost factor, impi = feature importance

4. Match Score Calculation

$$Score = \frac{\sum_{i=1}^{n} (factor_i \times weight_i)}{\sum_{i=1}^{n} weight_i} \times 100\%$$

Weighted average of all matching factors (grades, interests, preferences)

5. Cross-Validation Score

$$CV_{score} = \frac{1}{k}\sum_{i=1}^{k} Accuracy(fold_i)$$

k = 5 folds, provides robust accuracy estimate: 71.1% ¬± 4.3%
